# Personal Dotfiles

My collection of configuration files for various tools and applications.

## Contents

- **Awesome WM**: Dark theme configuration with system monitoring widgets
- **Alacritty**: Terminal emulator with multiple color themes

## Installation

```bash
git clone https://github.com/ViperBlackSkull/dotfiles.git
cd dotfiles
./install.sh
```

## Requirements

- Awesome window manager
- Alacritty terminal
- JetBrains Mono font (optional, will fallback to other monospace fonts)
- vicious library for Awesome widgets

## Configuration Details

### Awesome WM
- Dark theme optimized for low-light usage
- System monitoring widgets (CPU, Network)
- Custom keybindings
- Wallpaper support (includes custom Viper wallpaper)

### Alacritty
- Main configuration file: `alacritty.toml`
- Color themes included:
  - gruvbox.yml
  - nord.yml
  - tokyo-night.yml
  - alacritty.yml (default)
